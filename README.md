## 📋 О проекте

Nails Bot - инструмент, который поможет вести канал по маникюру. Бот может отправлять красиво оформленные посты, вести календарь записей прямо в канале, анализировать траты и покупки, отправлять подробные метрики и многое другое. Бот написан на чистом интузиазме в свободное от дневной рутины время, из за чего код не самый качественный и не самый правильный, но свою роль он выполняет

## ⚡️ Основные возможности

- 📊 **Система постов**
  - Автоматическое отправка поста в канал
  - Автоматическое отправление
  - Удобное динамичное создание бота прямиком в чате с ботом

- 📜 **Система записей**
  - Удобный контроль и ведение записей
  - Автоматическая отправка, обновление и закрепление сообщения с записями в канале
  - Автоматическое отслеживание устаревших дат
 
- 🛍️ **Система товаров**
  - Возможность составлять список купленных товаров и список товаров к покупке
  - Возможность вести учет купленных/некупленных товаров
  - Соотношений покупок с заработком

- 📈 **Аналитика**
  - Подробная статистика о покупках и тратах
  - Подробные метрики, анализирующие все доступные для бота данные
  - Метрики по разным типам
 
```⚠️Тут описан не весь функционал бота, его больше и он более объемный. Весь функционал можно посмотреть запустив бота у себя```

## 🚀 Установка и запуск
1. **Клонируйте репозиторий:**
```git clone https://github.com/lvd536/Nails-Content-Manager.git```
2. **Установите зависимости:** ```dotnet restore```
3. **Настройте конфигурацию:**
- Укажите токен вашего бота в ```Program.cs```
4. **Запустите проект:** ```dotnet run```
